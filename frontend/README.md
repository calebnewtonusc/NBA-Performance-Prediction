# NBA Performance Prediction - Frontend

Modern Next.js/React dashboard for NBA game predictions.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Charts:** Recharts
- **Icons:** Lucide React
- **API:** Axios

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Environment Variables

Create a `.env.local` file:

```
NEXT_PUBLIC_API_URL=https://nba-performance-prediction-production.up.railway.app
```

## Deployment

This app is configured to deploy to Vercel automatically from the GitHub repository.

## Features

- **Home:** API status dashboard
- **Game Predictions:** Predict NBA game outcomes
